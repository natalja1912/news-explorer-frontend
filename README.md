# Название проекта
**Сервис News-reader**
Фронтенд интерактивного приложения News-reader с регистрацией пользователей, где пользователи могут искать статьи по ключевым словам и сохранять выбранные статьи для дальнейшей работы.

## Технологии
 * React
 * React hooks
 * CSS Grid Layout
 * CSS Flex Layout
 * Адаптивная верстка
