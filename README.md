# Название проекта
**Сервис News-reader**
Интерактивное приложение News-reader с регистрацией пользователей, где пользователи могут искать статьи по ключевым словам и сохранять выбранные статьи для дальнейшей работы.

## Технологии
 * React
 * React hooks
 * Node JS
 * CSS Grid Layout
 * CSS Flex Layout
 * Адаптивная верстка
