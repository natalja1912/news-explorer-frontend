# Название проекта
**Сервис News-reader**
Бекенд интерактивного приложения с регистрацией пользователей, где пользователи могут искать статьи по ключевым словам и сохранять выбранные статьи для дальнейшей работы.

## Технологии
 * NODE JS
 * Express NODE JS
 * MongoDB
